# College Food Court Smart Ordering System 🍔

A web-based application designed to simplify food ordering in college campuses by reducing waiting time and managing food orders digitally.

## Features
- View food menu and availability
- Place online food orders
- User-friendly ordering interface
- Database management for orders and users
- Order status tracking
- UPI payment integration

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## Project Overview
This system helps students avoid long queues by allowing them to order food online from the college food court. It provides a simple and efficient way to manage food ordering and improve the overall campus dining experience.
